# Project 3: Around The U.S.

## Description

Around The U.S. is an interactive web page that allows users to share photos and engage with content from other users. Users can add new photos, remove their own photos, like photos posted by others, and make minor edits to their personal profile such as updating their name or description.
Technologies used:
HTML
CSS
Responsive design techniques

## Future Improvements

Store photos, likes, and profile data in a database
Allow users to comment on photos
